### Key Features
* **Semantic HTML Structure:** Uses `<header>`, `<main>`, `<section>`, and `<footer>` for better accessibility and SEO.
* **CSS Box Model:** implementation of:
    * **Padding:** To create breathing room within elements (e.g., skill tags, buttons).
    * **Margins:** To manage spacing between sections and layout elements.
    * **Borders:** Used for visual hierarchy (profile image, buttons).
* **Responsive Design:** Elements are spaced to look clean on different screen sizes.
